# Not an Expert Pack

A pack for 1.12.2. In development.

- Ice and Fire
- Extra Utils 2 / TMEL
- Ruins
- GregTech
- JourneyMap
- BetterNether / NetherEx
- Wireless Crafting Terminal and others
- BuildCraft
- Woot
- Apotheosis
- RailCraft
